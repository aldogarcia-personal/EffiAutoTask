# ADR-001 Sidebar Navigation

## Decisión

Se adopta **Sidebar Navigation Pattern** en lugar de **Top Navigation Pattern**.

## Motivo

La aplicación está diseñada como una **Productivity Web Application**, donde el número de secciones crecerá con el tiempo.

Este patrón permite:

- mayor escalabilidad de navegación
- organización jerárquica
- persistencia visual de secciones
- mejor compatibilidad con aplicaciones tipo workspace

Este modelo es utilizado en aplicaciones como:

- Notion
- ClickUp
- Linear
